# Changelog

All notable changes to the Pufferfish Portfolio Theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Initial release of Pufferfish Portfolio Theme
- Animated pufferfish cursor with bubble trail effect
- Responsive navigation with mobile menu
- Hero section with animated code block
- About section with technology list
- Project showcase with slider
- Certificates section
- Contact section with multiple contact methods
- Social media sidebar
- Smooth scroll animations
- Full Thai and English language support
- Remote theme support for easy deployment
- Comprehensive documentation (README, README_TH, QUICKSTART)
- Sample configuration file
- MIT License
